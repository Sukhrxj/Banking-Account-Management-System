# Banking Account Management System

A Java-based banking application that simulates real-world savings and chequing account operations using object-oriented programming and a graphical user interface.

## Features
- Create customer accounts
- Savings and chequing account management
- Secure deposits and withdrawals with overdraft prevention
- Auto-generated account numbers
- Graphical user interface for navigation and transactions

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Java Swing
- Eclipse IDE

## What I Learned
- Designing modular object-oriented systems
- Applying encapsulation and input validation
- Building multi-screen GUI workflows
- Translating real-world banking rules into code

## How to Run
1. Clone the repository
2. Open the project in Eclipse or any Java IDE
3. Run the main application file
